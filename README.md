# CPSC 380: Operating Systems

C/C++ projects exploring core operating system concepts, including process coordination, synchronization, CPU scheduling, and memory management.

## Featured Implementations

- **Thread Synchronization**
  - Implemented producer-consumer and reader-writer problems using POSIX threads, mutexes, semaphores, and condition variables

- **CPU Scheduling Algorithms**
  - Developed scheduling strategies including FCFS, SJF, Round Robin, and Priority scheduling
  - Simulated process execution and analyzed scheduling performance

- **Memory Management**
  - Built a contiguous memory allocation simulator using first-fit, best-fit, and worst-fit strategies
  - Implemented memory compaction and tracked fragmentation

- **Systems Programming**
  - Worked with low-level C/C++ to model OS behavior, including concurrency and resource management

## Projects

Each folder contains a single project focused on a specific OS concept:

- `P2_Thread_Sync` – Producer-consumer synchronization  
- `P3_ReaderWriter_Sync` – Reader-writer problem with synchronization  
- `P4_CPU_Scheduling` – Scheduling algorithm simulations  
- `P5_Contiguous_Mem` – Memory allocation and fragmentation  

## Tech Stack

- C / C++
- POSIX Threads (pthreads)
- Synchronization primitives (mutexes, semaphores, condition variables)
